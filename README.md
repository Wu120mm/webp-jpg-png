## 🛠️ 常见问题（FAQ）

### 1. **WebP 转换成 JPG 会损失质量吗？**
大多数工具在转换时会尽量保留原图质量，像 GitHub 的离线工具甚至接近无损。在线工具的质量通常也能满足日常需求。

### 2. **如何选择适合的工具？**
- 如果需要快速在线转换，推荐使用 iLoveIMG 或 Convertio。
- 如果需要批量转换或离线操作，GitHub 的 WebP-to-JPG 是不错的选择。

### 3. **这些工具是否支持 PNG 格式？**
大部分工具同时支持 WebP 转 PNG，具体功能可查看工具的官网说明。
